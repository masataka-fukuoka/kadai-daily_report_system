# kadai-daily_report_system

（２）成果物についての説明（６点）

＜１＞成果物の概要、特徴、機能

ログイン機能があり、ログイン時に使用したID・パスワードに該当する氏名で日報を書くことができます。
一覧ページには日報の書かれた日付,タイトルが表示され、詳細覧からは詳しい内容を見ることができます。
また、詳細覧から編集でき、その際更新日時が記載されます。
